# Complete Observability System (Metrics, Logs & Traces)

## 🎯 Objective
An integrated monitoring system with:
- **Prometheus** for metrics
- **Loki** for logs
- **Jaeger** for tracing
- **Grafana** dashboards for visualization

## 🧰 Tools Used
- Docker & Docker Compose  
- Prometheus  
- Grafana  
- Loki  
- Jaeger  
- Flask (Python)
